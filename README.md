# Tarefas App – Backend

API REST de tarefas com Node.js e Express.

## Requisitos
- Node.js instalado

## Instalação

```bash
npm install
```

## Como rodar

```bash
npm start
```

Servidor rodando em `http://localhost:3000`

## Rotas

| Método | Rota | Ação |
|--------|------|------|
| GET | /tarefas | Lista todas |
| POST | /tarefas | Cria nova |
| PUT | /tarefas/:id | Atualiza |
| DELETE | /tarefas/:id | Remove |
